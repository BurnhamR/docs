# Release 1.49.0

This release includes bug fixes, new features and stability improvements.

**Business Records**

With the new Business Search, also comes Business Records. Officers can create a criminal record for businesses.

**Documentation for values**

Work in progress: adding documentation for values. This should clear up some confusion on what each value is or does.

**Support selecting _all_ values in table**

When managing, admins who are able to delete values can now more easily delete all values to reset the table

**Editable SSN/VIN Toggle**

Disallow the ability for citizens to edit their SSN or vehicle's VIN. (CAD Feature - Optional Feature)
