# Release 1.56.0

This release includes minor bug fixes.

**Doctor Visits**

Allow EMS/FD to create doctor vists on a citizen. These are viewable via "Search Medical Records"

**Medical Records Management by citizen**

This feature can now be turned off. When disabled, only EMS/FD can manage citizens medical records.
