# Release 1.62.0

This release includes several UI improvements and stability improvements. Also includes a few new features

**Allow string badge numbers**

Since this update, officers and deputies are now also able to use letters in their badge numbers.

**User Defined Callsign**

Officers and deputies with the `SetUserDefinedCallsignOnOfficer` or `SetUserDefinedCallsignOnEmsFd` are now able to overwrite their callsign.

**Required Citizen Image (optional)**

You can now require a citizen image when creating a citizen. This can be toggled in the CAD Features tab inside CAD-settings.


**UI improvements**

A lot of UI improvements has been made in this release (Improved select, improved buttons, improved inputs, etc.).

**Bug fixes**

This release includes several bug fixes.
