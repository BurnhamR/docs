# Release 1.50.0

This release includes bug fixes and new features.

**Vehicle Trim Levels**

Define trim levels which can be used for Vehicle Values. Trim levels can then be set by citizens on their registered vehicle.

**Raw Webhooks**

SnailyCAD now supports Raw Webhooks. These allow you to simply set a webhook URL and handle the data yourself.

**Assign multiple units to call at a time**

You can now more easily assign multiple units to a 911 call at the same time.

**Improved documentation**

More documentation has been added for feature sets
