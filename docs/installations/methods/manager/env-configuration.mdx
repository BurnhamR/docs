# ENV Configuration

This guide will show you how to configure the ENV configuration.

:::tip
The "ENV" may also be referred to as "environment variables" or ".env file".
:::

:::info
The ENV Editor can be found at `CAD Toolbox` -> `ENV Editor`.
:::

The ENV configuration is a global configuration that is used to safely store credentials and other sensitive information.
