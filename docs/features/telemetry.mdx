# Telemetry

With telemetry, SnailyCAD sends anonymous error reports to our project managing software.
This makes it easier for us to fix bugs.
SnailyCAD only collects required data to correctly fix the incoming errors.

## How to disable telemetry

_Disabling telemetry will not have any drawbacks, it will just make it harder for me to fix bugs._

1. Open the `.env` file.
2. Add `TELEMETRY_ENABLED="false"`
3. [Correctly save the `.env` file](/docs/guides/how-to-properly-update-env)
4. Done.
