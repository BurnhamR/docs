# Supported Image types and URL's

SnailyCADv4 supports many image types and URL's. This page will list all supported image types and URL's.

### Supported Image types

- `image/png`
- `image/gif`
- `image/jpeg`
- `image/jpg`
- `image/webp`

### Supported image URL's

- `https://i.imgur.com/`
- `https://cdn.discordapp.com/`
