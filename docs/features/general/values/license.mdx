# License

This value defines the license of a citizen or vehicle. A license value has 4 types:

- `No type`: license has all types. It can be used anywhere a license is required.
- `License`: Can only be used when setting a license for a citizen such as drivers license, firearms license, etc.
- `Registration Status`: Can only be used when setting the registration status for a vehicle or weapon.
- `Insurance Status`: Can only be using when setting the insurance status for a vehicle or weapon.
