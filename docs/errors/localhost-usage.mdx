# Localhost Usage

This error will occur if you're using `localhost` as SnailyCAD's domain.

![localhost usage error](/localhost-usage.png)

## Possible ways to fix it

- Use your **LAN or WAN IP-address** for `NEXT_PUBLIC_PROD_ORIGIN` and `CORS_ORIGIN_URL`.
- Use **a domain name** for `NEXT_PUBLIC_PROD_ORIGIN` and `CORS_ORIGIN_URL`.
- Do not use `localhost` for `NEXT_PUBLIC_PROD_ORIGIN` and `CORS_ORIGIN_URL`.
